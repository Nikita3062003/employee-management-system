# 👩‍💻 Employee Management System (Django)

A web-based Employee Management System built using Django. This project allows users to manage employee records efficiently with features like adding, updating, and deleting employee details.

---

## 🚀 Features

- Add new employees
- Update employee details
- Delete employee records
- View all employees
- Simple and user-friendly interface
- Backend developed using Django

---

## 🛠️ Tech Stack

- Python
- Django
- HTML, CSS
- SQLite / MySQL
- Git & GitHub
